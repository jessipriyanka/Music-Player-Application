# -Music Player- 
CO7095 Group Project

## How to run 
1. Open PyCharm and import project folder.
2. Ensure Python 3.10+ interpreter is selected.
3. From terminal run: `python main.py`
4. To run tests:
   - Install pytest (if needed): `pip install pytest`
   - Run all tests: `pytest -q`
   - Run specific test: `pytest tests/blackbox/test_add_song.py -q`
